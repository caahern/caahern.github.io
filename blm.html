Black Lives Matter Website

On my semester exchange to the University of Hertfordshire I was given an assessment task focused on Web Design. This was different to any other web design task I had been given before, because we were limited to using Adobe Muse during production.

The brief was to create an immersive online experience for the Black Lives Matter movement. It was required to not only produce an immersive experience, but to be professional, interesting, interactive, clever and beautifully designed.

Although I struggled with Adobe Muse to begin with, I was quite happy with the end result with only a few changes I wanted to change using javascript or my own code if I had the chance. In saying this, there are a few minor glitches that I couldn't remove with the Muse program.

It was also a chance for me to learn a new Adobe suite which is always useful in the long run.
